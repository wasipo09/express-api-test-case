# To-Do List API Server 

## API Routes

* POST   /api/authenticate    
* POST   /api/register        
* POST   /api/task        C  
* GET    /api/task        R   
* GET    /api/task/:id    R 
* PUT    /api/task/:id    U   
* DEL    /api/task/:id    D  

## Test Case

Install all dependencies by
``` sh
npm install
```

Then run the server
``` sh
npm start
```

And have fun testing
``` sh
npm test
```

Be sure to check out the test case file in /testapi.js
